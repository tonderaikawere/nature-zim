## ğŸš€ Getting Started

### Prerequisites
- Python 3.7 or higher
- pip (Python package installer)

### Installation

1. **Clone or download the project**
   ```bash
   cd nature-zimbabwe
   ```

2. **Install Flask**
   ```bash
   pip install flask
   ```

3. **Run the application**
